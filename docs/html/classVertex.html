Edge* Vertex::path = nullptr
private

Pointer to the path edge of the vertex.

This pointer stores the path edge associated with the vertex during graph traversal algorithms. It points to the edge that leads to this vertex from its predecessor in the traversal. By default, it is initialized to nullptr.
